### 😮 ¿Qué es GitHub?
GitHub es una aplicación especializada para programadores, donde se guarda cada version de un archivo. Así no se pierde ningúna version anterior que haya existido de un mismo archivo
 
### 🎯 ¿Para qué me sirve?
GitHub me sirve para mantener mis códigos ordenados, y para que, si en algun momento tengo un error, pueda regresar a mi archivo anterior y empezar desde ahí y no desde cero
